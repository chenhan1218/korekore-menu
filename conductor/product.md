# Initial Concept

KoreKore - 日本餐廳菜單翻譯 AI 工具
Don't speak Japanese? Just point.

KoreKore 是一個協助台灣旅客在日本餐廳點餐的 Web App。使用者只需拍攝菜單圖片，AI 即可自動翻譯並生成「給店員看的原文點餐文字」。

🌟 核心功能
1. 菜單掃描 (Menu Scan)
用戶上傳或拍攝菜單圖片
支援 JPG、PNG 格式
實時處理反饋
2. AI 解析 (AI Processing)
呼叫 Gemini 1.5 Flash API 解析菜單
自動提取：菜名、台灣使用者習慣的繁體中文翻譯、簡單口感描述、價格
返回結構化 JSON 數據
3. 點餐介面 (Ordering UI)
將解析結果渲染成易讀的卡片清單
用戶可勾選想點的品項
實時預覽點餐內容
4. 生成點餐卡 (Order Card Generation)
根據勾選項目，生成包含敬語的原文點餐文字
例如：「すみません、これとこれをください...」
高對比度 UI，方便直接展示給店員

# Product Guide - KoreKore

## 產品願景 (Product Vision)
KoreKore 是一款專為台灣旅客設計的日本餐廳點餐支援工具。透過 AI 技術消除語言障礙，讓不懂日文的旅客也能輕鬆、自信地在日本餐廳享受美食。產品核心理念是「簡單拍照，輕鬆點餐」。

## 目標用戶 (Target Users)
* **台灣獨旅者 (Solo Travelers):** 需要快速、獨立完成點餐。
* **家庭或團體旅客 (Families/Groups):** 協助處理多人、多樣化的菜單選擇。
* **商務旅客 (Business Travelers):** 強求效率，需要在有限時間內精準點餐。

## 核心價值與目標 (Core Values & Goals)
* **極簡流程:** 奉行「拍照、勾選、展示」三步驟，追求在 10 秒內完成從分析到生成點餐文字的流程。
* **實戰可用性:** 確保「點餐卡 (Order Card)」介面在實際日式餐廳環境中易於被店員理解。
* **離線支援:** 提供離線快取功能，應對日本餐廳常見的網路連線不穩問題。
* **歷史追蹤:** 保留點餐紀錄，方便使用者回顧或再次點選喜愛的菜色。

## 關鍵功能 (Key Features)
* **Menu Scan (菜單掃描):** 支援 JPG/PNG 上傳，實時回饋處理狀態。
* **AI Processing (AI 解析):** 使用 Gemini 1.5 Flash API 解析菜單，提取菜名、繁體中文翻譯、口感描述與價格。
* **Ordering UI (點餐介面):** 卡片式清單設計，支援多品項勾選。
* **Order Card Generation (點餐卡生成):** 採用高對比度、超大字體的 UI，包含日文敬語與選中品項，方便直接展示給店員看。

## UI/UX 指導原則
* **語言規範:** 介面文字與錯誤訊息採中英雙語對照。對話與文件使用親切自然的繁體中文。
* **視覺風格:** 採用日式簡約風格 (Warm Muji-style)，給人溫暖且具食慾的感覺。
* **可讀性優先:** 點餐卡模式必須在各種光線條件下都能讓店員一眼看清。
